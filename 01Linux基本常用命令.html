• Linux基本命令
    • cp  /etc/passwd abc 将/etc/passwd 复制到当前目录并改名为abc
    • ll -a 查看当前目录所有详细信息包括隐藏
    • cd .. 回到上一级目录
    • cd Code 进入Code目录
    • cd ~ 回到当前用户的家目录
    • cd / 回到根目录
    • tree one 列出文件夹one下的所有子文件夹和文件（树形结构列出）
    • 绝对路径 是一个以根目录 / 为起点的完整路径，如： /home/shiyanlou/Code，可以类比 Windows 上的 C:\Users\电脑用户名\Desktop
    • 使用pwd命令获取当前目录的绝对路径
    • mkdir mycode 创建目录mycode 创建多级目录 mkdir -p one/two/three   然后进入到one目录输入tree可以查看目录结构
    • touch hello 创建一个名为hello的文件
    • cp hello one/two 复制文件hello到one/two目录中
    • mkdir test 新建目录test
    • cp -r test one/two 复制目录test 到 one/two目录中 (带参数r复制目录不带参数复制文件）
    • rm hello 删除文件hello
    • rm -r one 删除one这个目录(带参数r删除目录，不带参数删除文件 ）
    • 总结 cp和rm带参数r才能操作目录
    • mv test1 dir1 将test1文件移动到dir1目录下
    • mv test1 testbb 将文件test1重命名为testbb 因为testbb目录不存在所以是重命名操作
    • cp  /etc/passwd passwd 将/etc/passwd 复制到当前目录
    • cp  /etc/passwd abc 将/etc/passwd 复制到当前目录并改名为abc
    • cat abc 查看文件abc
    • cat -n passwd 带行号查看文件passwd的内容
    • cp  ../abc.py test 将上一级目录中的文件abc.py复制到目录test中
    • man cat 获取cat命令的详细帮助 进入man的页面后按空格翻页按q退出man
    • ls --help 查看ls的帮助
    • 这两个命令初学者未必看得懂，最佳途径是利用搜索引擎快速查找相关命令的使用
