  • 列表推导式
    • 列表解析式（List comprehension）或者称为列表推导式
    • 刚接触可能会复杂，其实列表推导式不仅可读性非常强，他所需要的代码量小，执行速度也最快的
    • 以下输出的三种等价
      • squares1 = []
      • for x in range(10):
      • squares1.append(x ** 2)
      • print(squares1)
      • squares2 = list(map(lambda x: x ** 2,range(10)))
      • print(squares2)
      • #此列表推导式等价于上面
      • squares3 = [x ** 2 for x in range(10)]
      • print(squares3)
    • 以下两种等价
      • print([(x,y) for x in [1,2,3] for y in [3,1,4] if x != y])
      • #等同于
      • combs = []
      • for x in [1,2,4]:
      • for y in[3,1,4]:
      • if x != y:
      • combs.append((x,y))
      • print(combs)
    • 列表推倒式的嵌套

      • z = [x + 1 for x in [x ** 2 for x in a]]

  • 集合的特性
    • # 集合是一个无序不重复元素的集
    • # 基本功能包括关系测试和消除重复元素
    • # 集合对象还支持union（联合） intersection（交）difference（查）和symmetric difference（对称差集）等数学运算
    • # 使用大括号函数set()来创建集合
    • # 要想创建空集合，必须使用set()而不是{}
    • 元素是否存在in
      • basket = {'apple','orange','pear','bann'}
      • print(basket)
      • print(type(basket))#集合类型
    • 集合经常操作两组数据
      • print('pear' in basket) #字符串pear在集合中存在吗 存在返回True
      • 集合主要用于两组数据的比较
      • print('\n')
      • ch1 = set('abracadabra')
      • ch2 = set('alacazam')
      • print(ch1)#集合会自动去掉重复的字符
      • print(ch2)#集合会自动去掉重复的字符
      • print(ch1-ch2)#ch1有 ch2没有的
      • print(ch1 | ch2)#存在于ch1或者ch2的
      • print(ch1 & ch2)#ch1和ch2都有的
      • print(ch1 ^ ch2)#存在于ch1或ch2但不同时存在的

    • # 用一对大括号{}创建一个空字典

      • print(type(a))#是个字典
    • 向字典中添加元素
      • #许多时候需要往字典中添加元素，首先要判断元素是否存在，
      • #不存在则创建默认值。如果在循环里执行这个操作，每次迭代都需要判断一次，降低程序性能。
      • # 我们可以使用 dict.setdefault(key, default) 更有效率的完成这个事情。
      • #setdefault() 字典中包含有给定键，则返回该键对应的值，否则返回为该键设置的值。
      • print('\n')
      • data8 = {10:20,'ac':"rb"}
      • data8.setdefault('abc','def')
      • print(data8)
      • data8.setdefault('abc',None)
      • print(data8)
      • print('\n')
      • dict = {'runoob': '菜鸟教程', 'google': 'Google 搜索'}
      • print('%s' % dict.setdefault('runoob',None))#键存在返回对应的值
      • print('%s' % dict.setdefault('Taobao','淘宝'))#键不存在返回设定的默认值
      • print('%s' % dict.setdefault('ak','47'))#查找dict中的键名ak是否存在，存在返回对应的值，不存在返回设定的值47
    • enumerate()遍历列表或者字典（或其他任何序列类型） 索引自动生成
