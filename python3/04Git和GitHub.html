04Git和GitHub
  • 创建文件并初始哈仓库
    • mkdir demo 创建目录demo
    • cd demo 进入目录
    • vim README.md 新建编辑自述文件
    • git init 将当前目录置于Git的管理之下（目录demo）
    • ls -a 有一个叫做.git的目录被创建了，说明仓库初始化成功
  • 提交代码
    • git add  README.md 将文件添加到本地的暂存区
    • 在提交之前必须先设定你的名字和Email （你GitHub网站的名字和注册使用的邮箱）
      • git config --global user.name "GitHubSoGood"
      • git config --global user.name "hullo@qq.com"
    • 提交到本地仓库
      • git commit -m "提交到本地仓库的备注信息"
    • 链接到GitHub仓库中
      • git remote add origin https://github.com/GitHubSoGood/haha.git
    • git push -f origin master
  • 验证用户名密码
    • 用户名是GitHub上的名字密码是你的登录密码
