14模块和包
  • python吸引人的一点是具有众多的模块可以使用自带模块https://docs.python.org/3/library/index.html
  • 第三方模块https://pypi.org/很多时候你都能找到合适的包帮你优雅的完成部分工作
  • 看模块是怎样工作的
    • 创建一个bars.py内容如下

  • def starbar(num): print('*' * num)def hashbar(num): print('#' * num)def simplebar(num): print('-' * num)​​​
  • 在cmd进入bars.py的路径并导入bars.py
    • python
    • import bars
    • bars.hashbar(10) #调用模块里面的函数
    • bars.simplebar(5)
  • 第二种方式，从模块中导入指定的函数可以这样做
    • from bars import simplebar,starbar
  • 第三种导入模块中的所有定义（不推荐）
    • from 模块名 import *
    • 例如from bars import *
  • 包
    • 包是模块的集合
      • 含有__init__.py文件的目录可以用来作为一个包，目录中的所有.py文件都是这个包的子模块包是模块的集合
      • import sound.effects.echo 这将加载子模块sound.effects.echo
      • from sound.effects import echo 这还将加载子模块echo
      • from sound.effects.echo import echofilter这将加载子模块echo中函数或变量echofilter
  • 默认模块
    • cmd下help()然后modules显示系统中安装的所有模块，也可以使用help()函数查找任何模块/类的文档
    • 如果你想知道字符串所有可用的方法，你可以这样做help(str)
